# Style Conventions

- classes and structures are denoted in Upper Camel Case

# operator overloads 

- vec2 + vec2 uses vec2::add
- vec2 - vec2 uses vec2::sub
- vec2 * vec2 uses vec2::dot
- float * vec2 uses vec2::scalarMult
